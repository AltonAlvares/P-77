Some things about Visual Studio Code and Google console screen



Visual Studio Code, also commonly referred to as VS Code, is a source-code editor made by Microsoft with the Electron Framework, for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.

The Console panel lets you run JavaScript code on your page. With the Console, you can test code snippets or debug your code. The Sources panel lets you view and edit your page's JavaScript and CSS code. With the Sources panel, you can set breakpoints, step through code, and see the values of variables.
